---
title: throttle
description: lodash의 throttle을 직접 구현한 함수
---

this에 타입 에러가 출력되시는 분들은 `tsconfig.json`에서 `noImplicitThis` 를 `false` 로 두시면 됩니다.

```ts title="utils/throttle.ts"
export function throttle(func: Function, wait: number) {
  let waiting = false
  return function () {
    if (!waiting) {
      func.apply(this, arguments)
      waiting = true
      setTimeout(() => {
        waiting = false
      }, wait)
    }
  }
}
```
