---

### 웹 플랫폼

- 웹을 구축하고 실행하기 위한 기술과 환경을 제공하는 플랫폼
- 웹 브라우저와 관련된 기술, 웹 서버, 데이터베이스 등 다양한 기술과 소프트웨어의 집합을 포함
- 웹 브라우저 환경(Chrome, Firefox, Safari 등), 웹 서버(NGINX, Apache 등), 데이터베이스 시스템(MySQL, PostgreSQL 등) 등

### 웹 애플리케이션

- 웹 플랫폼 위에서 동작하는 응용 프로그램
- 웹 브라우저를 통해 사용자들이 웹을 통해 서비스를 이용하고 상호작용하는 소프트웨어
- 클라이언트와 서버 사이에서 데이터 교환과 사용자 인터페이스를 제공하며, 대부분의 로직은 서버 측에서 처리
- 이메일 서비스, 쇼핑 웹사이트, 소셜 미디어 플랫폼 등
